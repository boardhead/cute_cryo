CUTE Cryostat Position Control Help

The CUTE cryostat is mounted on a set of 3 dampers to isolate it from external sources of vibration. This position system monitors and controls the positions of these dampers to keep the cryostat in a fixed location.

This documentation explains the elements of the html interface to this system.

Motor speeds

These indicate the speed in steps/sec and direction of each of the 3 control motors. Positive motor speed drives the damper upwards.

Active Control

Enable or disable the active position control. When activated, the control system will attempt to keep all dampers at a position of 1.0 ± 0.1 mm.

Air Pressure

Measured ambient air pressure in the mine. This value is used in the calculation for the amount of weights to add.

Damper positions

Position in mm of the three dampers. The target position is 1.0 ± 0.1 mm.

Add weights

These indicators are used when adding weights to balance the cryostat. When the system is active and under vacuum, weights should be added (+ve values) or removed (-ve values) from each damper until these indicators are centered.

Damper position history

This is a 10-minute history of the three damper positions. The most current reading is at the left edge of this plot, and the points slide to the right as time advances.

Comment

Type a comment in this field then press RETURN to enter the comment in the log. This may be used as a simple chat feature to communicate with other clients connected to the position control system.

The up/down arrows on the keyboard may be used to recall previous comments.

The comment field supports a set of commands beginning with a "/":

/active [on|off|start]Enable/disable active control. "start" starts motor even if damper is within tolerance.
/avr# CMDSend AVR command (# = AVR number). eg. "/avr0 help".
/calShow raw and calibrated readings from ADAM-6017 ADC.
/helpShow this list of commands.
/listList connected AVR's.
/log MSGEnter a log message. This is the same as entering a comment in the log except that the message is not quoted.
/name [WHO]Get/set client name.
/verbose [on|off]Get/set verbose state. Raw Adam readings are logged once per second while verbose.
/whoList connected clients.

Pressing ESC in this field stops all motors when active control is off.
